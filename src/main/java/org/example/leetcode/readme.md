## 搜索二叉树：左树的节点都小于根节点，右树的节点都大雨根节点
    中序遍历，依次升序，一定是搜索二叉树
    

    完全二叉树（cbt）：除了最后一层，每一层都是满满的 2 最后一层的节点，尽可能的靠左排列
    满二叉树：n = 2 ^l - 1 最大深度 l
    平衡二叉树（avs）:左树和右树的高度差不能超过1
